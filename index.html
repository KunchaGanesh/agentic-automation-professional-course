Agentic Automation Professional

Design, build, and manage advanced agentic automation solutions in UiPath — combining the strengths of AI-powered agents with robust enterprise RPA. Select the right agent types, ground AI in trustworthy business context, build coded agents, and orchestrate asynchronous, human-in-the-loop workflows in Maestro.

Course objectives

What you will be able to do

By the end of this course, you will be able to:

  • Select the appropriate UiPath agent type and identify strong agentic automation use cases.
  • Design agent stories, prompts, workflows, and trustworthy Context Grounding strategies.
  • Develop, secure, deploy, and manage coded agents using Python, UiPath SDKs, and Orchestrator.
  • Evaluate, trace, debug, monitor, and continuously improve agent performance.
  • Build resilient asynchronous and human-in-the-loop processes using UiPath Maestro.
  • Integrate agents with enterprise systems through API workflows, webhooks, events, and Autopilot.
